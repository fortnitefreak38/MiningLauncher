# MiningLauncher

Electron-basierter Miner-Launcher mit Web-UI, Telegram/Discord-Benachrichtigungen und Auto-Update.

## Features

- Miner starten/stoppen über Web-UI
- Live-Hashrate-Überwachung
- Telegram- & Discord-Benachrichtigungen
- Mehrere Miner gleichzeitig verwalten
- Auto-Update (GitHub Releases)
- Läuft im Tray (System Tray)

## Dev

```bash
npm install
npm start
```

## Build

```bash
npm run build
```
